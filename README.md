# 🕒 工業計時儀 Pro (V9.0 巔峰版)

這是一款專為 Windows 設計的高精度工業風計時工具，現已全面進化。

## 🌟 V9.0 全新特色
* **夜間模式切換**：支援一鍵切換深色/淺色工業主題，適應不同辦公環境。
* **自定義鈴聲**：新增多組工業警示音效，提醒更到位。
* **完全免費**：本工具目前提供的所有服務皆為完全免費使用。
* **智能校準邏輯**：防止輸入超標數字（如 66:66 將自動修正為 23:59）。
* **台北時區同步**：自動偵測並顯示台北標準時間 (CST)。

## 🚀 下載與執行
1. **前往網頁**：透過 GitHub Pages 連結進入下載頁。
2. **立即使用**：免安裝綠色版，下載 `.exe` 後雙擊即可啟動。
